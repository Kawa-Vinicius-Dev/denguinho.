Criei esse app para tirar nossos combinados do papel e transformar nossa rotina em uma disputa saudável. A ideia foi unir o útil ao agradável: praticar minhas habilidades com React + Supabase e, de quebra, dar um up no nosso dia a dia.

️O que o app faz?
Check-in Instantâneo: Marcou, contou. O placar atualiza na hora no celular do outro (Realtime).

Controle de Metas: Dá para criar desafios diários, semanais ou mensais.

Histórico Manual: Esqueceu de marcar ontem? Tem uma grade para recuperar os pontos com trava de segurança para não clicar errado.

Placar Eterno: Uma contagem geral de quem está ganhando a brincadeira no mês.

Vibração: Feedback tátil no celular ao completar uma tarefa (sensação de "feito").

O que tem por baixo do capô?
Vite + React: Para o app abrir voando.

Tailwind CSS: Visual moderno, limpo e com Dark Mode caprichado.

Supabase (Postgres): Onde a mágica do tempo real e do banco de dados acontece.

Deploy na Vercel: Hospedagem rápida e segura.

️Por que eu fiz isso?
Muito mais do que uma lista de tarefas, eu quis construir uma ferramenta nossa. Foquei em criar uma interface fluida e intuitiva que desse vontade de usar todo dia. É tecnologia aplicada para fortalecer a nossa parceria e constância.
